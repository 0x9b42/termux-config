# language
English: "Where is the nearest coffee shop?"

Russian:

Где ближайшее кафе? (Gde blizhaysheye kafe?)

Где = "where."

Ближайшее = "nearest" (neuter form, agreeing with кафе).

Кафе is neuter and indeclinable.


Spanish:

¿Dónde está la cafetería más cercana?

¿Dónde está…? = "Where is…?"

La cafetería = "the coffee shop" (feminine).

Más cercana (nearest) agrees with la cafetería.


Chinese:

最近的咖啡店在哪里？ (Zuìjìn de kāfēidiàn zài nǎlǐ?)

最近的 = "nearest" (最近 = recent/close, 的 makes it an adjective).

咖啡店 = "coffee shop."

在哪里？ = "is where?" (different word order from English).


French:

Où est le café le plus proche ?

Où est…? = "Where is…?"

Le café = "the coffee shop" (masculine).

Le plus proche = "the nearest," agreeing in gender with le café.


Arabic:

أين أقرب مقهى؟ (Ayna aqrab maqha?)

أين = "where."

أقرب = "nearest" (doesn’t change gender here).

مقهى = "coffee shop" (masculine).


Korean:

가장 가까운 커피숍이 어디예요? (Gajang gakkaun keopisyop-i eodiyeyo?)

가장 = "most" (superlative).

가까운 = "near/close" (adjective).

커피숍 = "coffee shop" (loanword from English).

이 어디예요? = "is where?"


Japanese:

一番近いカフェはどこですか？ (Ichiban chikai kafe wa doko desu ka?)

一番 = "most" (superlative).

近い = "near" (adjective).

カフェ = "café" (loanword).

はどこですか？ = "is where?"


German:

Wo ist das nächste Café?

Wo ist…? = "Where is…?"

Das Café = "the café" (neuter).

Nächste = "nearest" (neuter form, agreeing with das Café).
